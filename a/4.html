Deadline

18:00 04.05.2020

Description

Implement new features of your web site:

  • A user can book property and cancel the booking

    Inside the property’s profile a user must press the button “Book” in order to book the property. After pressing the button, the number of available rooms or the overall property availability changes. If the property if fully booked it is removed from the general list. The user can also cancel his booking by pressing the respective button.

  • Add missing property data fields from the list 

    name;
address;
quality (stars);
users’ rank
(from 0 to 10);
opportunities list;
photos;
closest landmarks;  // a list of closest sights
rent price;
payment types;
reviews section

  • User can review his bookings in a separate page

    Every user can check his list of bookings ("My bookings")

  • Deploy a model for the property recommendation system.
