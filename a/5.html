Assignment #5. Search and recommendations (10 points)

Deadline

18:00 18.05.2020

Description

Implement new features of your web site:

  • Implement a search function with the following parameters 
    location;
check-in/out dates;
sleep places quantity;
age specification (adults/children);
rent price

    Search opportunity must be implemented in order to filter property. Age statuses: "No children", "Only adults", "No restrictions"

  • A user must be able to rate the property.

    After the booking is expired, the user can rate (0-10 rating) along and comment the property. Average rating among all users is displayed

  • Recommended booking places according to the user’s activity.

    On the basis of favorite booking stats a user is provided with certain recommendations.
